## Run Locally

Clone the project

```bash
  git clone https://github.com/songloimr/ReactNativeASM.git
```

Go to the project directory

```bash
  cd ReactNativeASM
```

Install dependencies

```bash
  yarn install
```

Start the app

```bash
  yarn android
```
